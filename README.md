# connect_four

Rust implementation of a perfect AI Connect Four solver.

[//]: # (badges)

[//]: # (visuals)

## Installation

```shell
git clone git@github.com:yoonthegoon/connect_four.rs.git
cd connect_four.rs
cargo build
```

## Usage

```shell
# usage: connect_four [POSITION]
connect_four 4453
```
